FlexBox

Pada module flexbox ini dapat kita bagi menjadi dua kelompok, yaitu layouting properties dan alignment properties

Layouting Properties

Properti-properti ini berfungsi untuk mengatur tataletak komponen-komponen element

  1. Flex(<'flex-grow'> <'flex-shrink'> <'flex-basis'>)

    flex merupakan shorthand dari property CSS yang dapat terdiri dari 3 properti yaitu flex-grow, flex-shrink dan flex-basis

    Property flex dapat berisikan 1,2 dan 3 value, tiap value memiliki tugasnya masing-masing. ada yang menentukan tingkat pelebaran element(flex-grow), ada yang mengatur tingkat penyusutan element(flex-shrink) dan ada pula yang mengatur lebar awal element(flex-basis) 

       
/* One value: flex-grow */
flex: 1;

/* Two values: flex-grow | flex-basis */
flex: 1 30px;

/* Two values: flex-grow | flex-shrink */
flex: 2 2;

/* Three values: flex-grow | flex-shrink | flex-basis */
flex: 2 2 10%;

    1. flex grow

      Flex grow berfungsi untuk menentukan lebar child dari sebuah container flex.

      Arah lebar dari flex-grow mengikuti main axis atau arah sumbu yang di atur oleh value flex-direction. jika flex-direction:row; maka flex-grow melebar secara horizontal, jika flex-direction:column; maka flex-grow melebar secara vertikal

         
  /* < number > values */
  flex-grow: 3;
  flex-grow: 0.6;

    2. flex-shrink

      Kalo flex-grow berfungsi untuk mengatur tingkat pelebaran, maka sebaliknya flex-shrink berfungsi untuk mengatur tingkat penyusutan.

      Sama seperti flex-grow, arah penyusutannya mengikuti main exis(arah sumbu), jika value flex-direction:row; maka penyusutan secra horizontal dan sebaliknya

         
  /* < number > values */
  flex-shrink: 2;
  flex-shrink: 0.6;

    3. flex-basis

      Flex-basis berfungsi untuk menentukan pada ukuran awal dari element dan pada ukuran tersebut property flex-grow dan flex-shrink mulai bekerja

        

  /* Specify <'width'> */
flex-basis: 10em;
flex-basis: 3px;
flex-basis: 50%;
flex-basis: auto;

  2. flex-flow(<'flex-direction'> <'flex-wrap'>)

    flex flow merupakan shorthan atau gabungan dari flex-direction dan flex-wrap

     
  /* flex-flow: <'flex-direction'> and <'flex-wrap'> */
flex-flow: row nowrap;
flex-flow: column wrap;
flex-flow: column-reverse wrap-reverse;

    1. flex-direction

      Flex direction berfungsi untuk memberikan arah main exis(sumbu) pada element yang berada di dalam parent flex.

      row / row-reverse => horizontal

      column / column-reverse => vertikal

             
      /* The direction text is laid out in a line */
      flex-direction: row;
      
      /* Like < row >, but reversed */
      flex-direction: row-reverse;
      
      /* The direction in which lines of text are stacked */
      flex-direction: column;
      
      /* Like < column >, but reversed */
      flex-direction: column-reverse;

    2. flex-wrap

      Property yang berfungsi untuk menentukan apakah flex items dipaksakan dibungkus pada satu line atau dapat di bungkus dalam multiple line

            
    
      flex-wrap: nowrap; /* Default value */
    flex-wrap: wrap;
    flex-wrap: wrap-reverse;

  3. order

    Property ini berfungsi untuk mengatur urutan posisi flex item
