CSS Layout Dasar

  1. display

    setiap element pada HTML berada dalam sebuah kotak. properti display pada css mengatur perilaku kotak tersebut

    properti display memilik beberapa value:

    • inline

      element yang secara default tidak menambahkan garis baru ketika dibuat

      lebar dan tingginya sebesar kontent yang ada didalamnya

      Kita tidak bisa mengatur width dan height nya

          b, strong, i, em, span, a, sub, sup, button, input, label, select, textarea

    • inline-block

      Secara default tidak ada element HTML yang memiliki nilai display inline-block

      Harus secara manual mengubah nilai properti tersebut

      Perilaku dasarnya sama seperti inline, perbedaannya elemen inline-block dapat di atur width dan height

    • block

      Element yang secara default menambahkan garis baru ketika dibuat

      Secara default lebarnya akan memenuhi layar browser jika tidak di atur 

          p, div, h1-h6, ol, ul, li, form, hr

    • none

      diguankan untuk menghilangkan element

  2. dimensi

    width, untuk mengatur lebar

    height, untuk mengatur tinggi

    satuan yang bisa kita set yaitu: px, %, in, cm, mm, pt, pc

  3. overflow

    Berfungsi untuk mengatur prilaku content pada sebuah element

    terdapat beberapa value pada properti overflow:

    1. visible

      Value visible(terlihat) adalah value default

    2. auto

      Apabila content overflow terhadap element maka akan memunculkan scroll, akan tetapi jika konetnnya cukup maka tidak memunculkan overflow

    3. hidden

      Apabila content melebihi element pembungkusnya maka content yang overflow tersebut akan di sembunyikan/hidden

    4. scroll

      Apabila content overflow atau tidak scroll akan terus muncul

  8. float

    Folat merupakan properti CSS untuk mengatur posisisebuah element

    normal flow, seperti yang sebelumnya di pelajari, ketika element inline kita ubah menjadi block maka element tersebut dapat diubah dimensinya(width & height). itu adalah normal-normal saja(normal flow)

    out-of flow, ketika sebuah element diberikan css properti float (left & right)

    properti float memiliki tiga value:

    • none

      nilai default

    • left

      element akan dipaksa ke kiri

    • right

      element akan di paksa ke kanan

  9. position

    terdapat empat value yang dapat digunakan pada properti position:

    • static

      static merupakan value default dari properti position

      menggunakan value selain static/non-static(relative, absolute dan fixed) akan membuat element tsb menjadi berbeda dimensi dengan element yang memiliki value static. Sehingga element yang memiliki value selain static(relative, absolute dan fixed) dapat mengakses properti top, left, bottom, right atau inset:"top" | "right" | "bottom" | "left";

    • relative

    • absolute

    • fixed

      Element yang menggunakan value fixed, dia akan bergerak relatif terhadap window browser dan tidak memperdulikan parent-nya walaupun parent bersifat non-static

    • Sticky

      position sticky membuat elemen berperilaku seperti relative sampai mencapai batas tertentu selama pengguliran, di mana elemen tersebut menjadi fixed dan tetap pada posisi yang ditentukan.

      Properti top, right, bottom, dan left sangat penting karena menentukan batas di mana elemen beralih dari perilaku relative menjadi fixed.

      Misalnya, dengan mengatur top: 0;, elemen akan menempel pada bagian atas halaman saat mencapai bagian atas viewport.

  18. animation

    1. Animation

      peroperti animation sebenarnya adalah shorthand dari CSS animation, Selengkapnya:

      > Animation Syntax

  /* @keyframes/animation duration | easing/timing function | delay | iteration-count | 
  direction | fill-mode | play-state | name */
  animation: 3s ease-in 1s 2 reverse both paused slidein;

  /* @keyframes/animation duration | easing-function | delay | name ("WAJIB") */
  animation: 3s linear 1s slidein;

  /* two animations */
  animation:
    3s linear slidein,
    3s ease-out 5s slideout;

animation duration  : > Durasi keyframes/animation 
                      > animation-duration: 3s;
timing functoin     : > easing/timing function 
                      > animation-timing-function: ease-in-out;
delay               : > Delay sebelum animasi dijalankan 
                      > animation-delay: 2s;
iterration          : > angka | infinite (pengulangan berapa kali?) 
                      > animation-iteration-count: 1.5;
direction           : > normal (berjalan dari awal sampai akhir) 
                      > reverse (berjalan dari akhir sampai depan)
                      > alternate (berjalan dari awal sampai akhir dan kembali dari akhir sampai awal)
                      > alternate-reverse (berjalan dari akhir sampai awal dan kembali dari awal sampai akhir)
                      > animation-direction: alternate;
fillmode            : > untuk menentukan akhir dari transisi 
                      > none (nilai default, berhenti di keadaan awal object)
                      > forwards (berhenti di keadaan akhir animasi)
                      > backwards (kaya none)
                      > animation-fill-mode: forwards;
play state          : > Membuat animasi dijalankan(running) atau berhenti (paused) 
                      > animation-play-state: paused;
name                : > nama animation 
                      > animation-name: myAnimation;


// Pembuatan animation
    @keyframe [keyframe-name] {
        from {
            [css property]
        }
        to {
            [css property]
        }
    }

    2. Animation Timeline

      animation-timeline adalah properti CSS yang berfungsi untuk menentukan timeline yang digunakan untuk mengontrol perubahan animasi, berikut beberapa type timeline yang dapat di set pada properti animation-timeline

      1. scroll-timeline

        scroll-timeline adalah shorthand dari propert CSS yang digunakan untuk mendefinisikan sebuah nama scroll progress timeline

        > Penggunaan 
  Dalam penggunaannya, membutuhkan beberapa property lain sehingga progres scroll element 
  dapat dijadikan timeline animasi:
  1. timeline-scope
    Properti ini berfungsi untuk menentukan nama timeline yang akan kita buat 
    (harus menggunakan dua tanda "--".) dan properti ini berfungsi untuk
    menentukan scope element mana saja yang dapat menggunakan timeline ini.
    (!!Baca notes pada source code!!)
    contoh: "timeline-scope:--myScroller;"

  2. scroll-timeline
    Properti ini berfungsi untuk menjadikan element yang memiliki properti ini Apabila
    memiliki scroll, maka progres scroll tersebut akan menjadi timeline animasi
    contoh: scroll-timeline:--myScroller;

  3. animation-timeline
    Element yang memiliki properti animation-timeline, maka jalannya animasi pada element
      terebut mengikuti progres scroll element terkait.
      contoh: animation-timeline:--myScroller;
