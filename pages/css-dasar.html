  1. Apa itu CSS?

  2. Penempatan CSS

    terdapat tiga penempatan css yang bisa dilakukan

    1. Embed / Internal

      dengan menggunakan tag style, didalam tag head pada file HTML

    2. Inline

      di letakan di dalam sebuah tag yang ingin diberikan style CSS, dengan menambahkan attribute style=""

    3. Eksternal

      menggunakan file terpisah dari file HTML dengan extention .css, contoh: index.css. untuk menghubungkan file HTML dengan file css eksternal tersebut dapat menggunakan tag link dengan attribute rell="stylesheet" dan src=""(menuju file CSS yang di tuju)

  3. font styling

    beberapa properti yang dimiliki oleh CSS dalam mengatur font:

    

shortcut:
  /* font-size font-family */
font: 1.2em "Fira Sans", sans-serif;

/* font-size/line height font-family */
font: 1.2em/2 "Fira Sans", sans-serif;

/* font-style font-weight font-size font-family */
font: italic bold 1.2em "Fira Sans", sans-serif;

/* font-stretch font-variant font-size font-family */
font: ultra-condensed small-caps 1.2em "Fira Sans", sans-serif;

/* system font */
font: caption;

    1. font-family

      mengatur jenis font(menggunakan font-family: 'Courier New', Courier, monospace) yang akan digunakan

    2. font-size

      mengatur ukuran font(menggunakan font-size:30px)

    3. font-weight

      mengatur ketebalan font(menggunakan font-weight:900;)

    4. font-variant

      mengubah font menjadi small caps(menggunakan properti font "small-caps")

    5. font style

      mengubah font menjadi bercetekmiring(menggunakan font-style: italic)

    6. line height

      mengatur spasi antar baris(menggunakan line-height:20px)

  5. Background

    Properti background merupakan shorthand yang berfungsi untuk mengatur semua properti background style menjadi satu properti, seperti bg-color, bg-image, bg-size, bg-repeat, bg-position:

      
  /* bg-image | bg-attachment | bg-repeat | bg-clip | bg-origin | bg-image */
  background: url("image.jpg") fixed no-repeat content-box padding-box, linear-gradient(to bottom, #00f, #fff)  ;

    1. background-attachment

      Properti background-attachment berfungsi unutk mengatur apakah posisi background-image fixed terhadap viewport ataukan scroll bersamaan dengan block yang berada didalamnnya

    2. background-color

      berfungsi untuk mengatur warna background

    3. background-image

      mengatur gambar yang akan digunakan pada background

      cara penggunaannya seperti berikut:

          background-image: url(./photo-example.png);

      Secara default background image akan me-repeat terhadap sumbu x (horizontal) dan sumbu y(vertikal). untuk mengatur pperulangan/repeat, kita dapat menggunakan properti background-repeat:..;

    4. background-repeat

      mengatur jenis perulangan gambar pada background

      terdapat beberapa value pada background-repeat:

          
    background-repeat: repeat;    //melakukan perulangan sumbu x dan y
    background-repeat: repeat-x;  //melakukan perulangan sumbu x saja
    background-repeat: repeat-y;  //melakukan perulangan sumbu y saja
    background-repeat: no-repeat; //tidak melakukan perulangan

      !!JIKA!! pada propertibackground-repeat menggunakan value no-repeat, maka kita memiliki akses dalam menggunakan properti background-position

    5. background-position

      mengatur posisi gambar pada background

          
    background-position: top left;  // x | y
    background-position: 10px 20px
    background-position: 10% 20%
  
    short hand:
      > background: color url() position repeat;
      > background: coral url(./ex.png) top left no-repeat;

  6. selectors

    Digunakan untuk mengenali sebuah element HTML yang akan diberikan style

    Terdapat banyak selector yang bisa di gunakan sesuai kebutuhan:

    1. Selector Dasar

      > Universal Selector (*):
  Memilih semua elemen pada halaman.
  * {
    /* Rules */
  }

> Element Selector:
  Memilih elemen berdasarkan tagnya.
  h1, h2, h3 {
    /* Rules */
  }

> Class Selector (.):
  Memilih elemen berdasarkan kelasnya.
  .my-class {
    /* Rules */
  }

> ID Selector (#):
  Memilih elemen berdasarkan ID-nya.
  #my-id {
    /* Rules */
  }

    2. Selector Attribute

      > Attribute Selector ([attribute="value"]):
  Memilih elemen berdasarkan atributnya.
  label[for="attribute-selector"] {
    /* Rules */
  }
  
  < label for="attribute-selector">Username:< /label>
  < input type="text" name="" id="attribute-selector">
  < label for="attribute-sele">Full Name:< /label>
  < input type="text" name="" id="attribute-sele">

  nb: penggunaannya bisa seperti ini: label[for]{...}, jadi tidak
  tidak menggunakan value attributnya, cukup attribute name-nya saja
  kalo emang mau menuju element yang memiliki attribute "for"


> Attribute Starts With Selector ([attribute^="value"]):
  Memilih elemen yang memiliki atribut dengan value awal 
  yang sesuai dengan selector
  [class^="lower"] {
    /* Rules */
  }

  < p class="upper-par-weight-normal">Lorem ipsum dolor sit.< /p>
  < p class="lower-par-weight-normal">Lorem ipsum dolor sit.< /p>
  < p class="lower-par-weight-normal">Lorem ipsum dolor sit.< /p>
  < p class="par-weight-normal">Lorem ipsum dolor sit.< /p>


> Attribute Ends With Selector ([attribute$="value"]):
  Memilih elemen yang memiliki atribut dengan value akhir 
  yang sesuai dengan selector.
  [class$="normal"] {
    /* Rules */
  }
  
  < p class="upper-par-weight-normal">Lorem ipsum dolor sit.< /p>
  < p class="lower-par-weight-normal">Lorem ipsum dolor sit.< /p>
  < p class="lower-par-weight-normal">Lorem ipsum dolor sit.< /p>
  < p class="par-weight-normal">Lorem ipsum dolor sit.< /p>


> Attribute Includes Word Selector ([attribute~="value"])
  Selector ini memilih elemen di mana nilai atribut tersebut mengandung kata 
  tertentu di antara spasi. Ini berfokus pada kata-kata yang dipisahkan oleh 
  spasi dalam nilai atribut.
  [class~="button"] {
    /* rules */
  }
  < button class="button-submit">< /button>
  < button class="button btn-submit">< /button>
  nb: penggunaan selector ini sama persis kaya penggunaan class selector


> Attribute Contains Selector ([attribute*="value"]):
  Selector ini memilih semua elemen di mana nilai atribut tersebut 
  mengandung substring yang sesuai dengan nilai yang ditentukan.
  [class*="center"] {
    /* Rules */
  }

  < p class="lower-par-weight-normal-center">Lorem ipsum dolor sit.
  < p class="lower-par-end-weight-normal">Lorem ipsum dolor sit.
  < p class="lower-par-center-weight-normal">Lorem ipsum dolor sit.
  < p class="lower-par-end-weight-normal">Lorem ipsum dolor sit.
  

> Attribute Represents a Dash-Separated List of Values Selector (["class|="align"]):
  Selector ini digunakan untuk memilih elemen yang memiliki atribut tertentu di mana nilai 
  atribut tersebut mengandung kata kunci tertentu baik yang dipisahkan oleh tanda hubung (-) 
  atau hanya satu kata kunci tersebut pada value attribute
  [class|="card"] {
    /* rules */
  }

  < div class="card">
    < div class="card-header">< /div>
    < div class="card-body">< /div>
  < /div>

    3. Selector Kombinator

      > Descendant Selector (Whitespace):
  Memilih elemen yang berada di dalam elemen lain, tidak hanya langsung child.
div p {
  /* Rules */
}

< div>
    < article>
      < p>lorem< /p>
    < /article>
    < p>lorem< /p>
< /div>
< p>lorem< /p>


> Child Selector (>):
  Memilih elemen yang menjadi child langsung dari elemen parentnya.

div > p {
  /* Rules */
}

< div>
    < article>
      < p>lorem< /p>
    < /article>
  < p>lorem< /p>
< /div>
< p>lorem< /p>


> Adjacent Sibling Selector (+):
  Memilih elemen yang merupakan saudara sejajar (setelahnya) langsung (sibling) dari
  elemen sebelumnya. !! Hanya satu saudara sejajar yang dipilih
div + p {
  /* Rules */
}

< div>
    < article>
      < p>lorem< /p>
    < /article>
    < p>lorem< /p>
< /div>
< p>lorem< /p>
< p>lorem< /p>


> General Sibling Selector (~)
  Sama seperti Adjacent Sibling Selector, akan tetapi yang dipilih semua
  saudara sejajarnya. !! Hanya saudara sejajar setelahnya yang dipilih
div ~ p {
  /* Rules */
}

< div>
    < article>
      < p>lorem< /p>
    < /article>
    < p>lorem< /p>
< /div>
< p>lorem< /p>
< p>lorem< /p>

    4. Selector nth

      > Nth Child Selector (:nth-child()):
  Memilih elemen berdasarkan posisinya sebagai child ke-n dari elemen parent.
  li:nth-child(2) {
  /* Rules */
}

> First-of-type, Last-of-type, 
  Nth-of-type Selector (:first-of-type, :last-of-type, :nth-of-type()):
  Memilih elemen berdasarkan tipe elemennya.
p:first-of-type {
  /* Rules */
}

  7. pseudo class

    Pseudo class adalah sebuah class yang akan aktif pada keadaan tertentu. terdapat beberapa tipe pseudo class:

    pseudo-class yang berhubungan dengan link

    1. :link

      style default pada sebuah link(anchor)

    2. :hover

      style yang akan aktif ketika krusor berada di atas element / link

    3. :active

      style yang aktif jika sebuah link / element di click

    4. :visited

      style yang akan aktif jika sebuah link sudah pernah di kunjungi sebelumnya(menggunakan browser yang sama)

    pseudo-class yang berhubungan dengan posisi element

    1. :first-child

      memilih element/child pertama dari sebuah parent

    2. :last-child

      memilih element/child terakhir dari sebuah parent

    3. :nth-child(n)

      memilih element ke-n dari sebuah parent, "n" memiliki beberapa pola

      • Pola berurut (1), (2), (3)...

      • Pola kelipatan (2n), (2n+2), (3n-2)

      • Genap(event)

      • Ganjil(odd)

  8. specificity

    setiap selector memiliki berat yang berbeda. berat tersebut menentukan seberapa spesifi sebuah element dapat dipilih oleh selector

    1. Element(p, h1-h6, div, span) => 0001

    2. class(.card; .nav_bar;) => 0010

    3. id(#form-input; #note-id_111;) => 0100

    4. Inline(< div style...>< /div >) => 1000
